## Installation

```bash
# clone the repo
$ git clone https://github.com/ukwatte/DLCoronaVirusData-CSV.git

# install the requirements
$ pip3 install -r requirements.txt
```

## Usage
```
usage: python3 DLCoronaDataCSV.py /Users/USERNAME/output.csv
```
